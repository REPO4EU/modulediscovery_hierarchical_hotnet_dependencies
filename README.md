# modulediscovery_hierarchical_hotnet_dependencies

Docker Container for the Python dependencies used for the Hierarchical HotNet tool of the [diseasemodulediscovery](https://github.com/nf-core/diseasemodulediscovery.git) pipeline 

The packages in the [environment.yml](./environment.yml) are different then stated in the [Hierarchical HotNet repository](https://github.com/raphael-group/hierarchical-hotnet.git), to ensure compatibility with the pipeline. 

## Citation
Hierarchical Hotnet publication: Reyna MA, Leiserson MDM, Raphael BJ. Hierarchical HotNet: identifying hierarchies of altered subnetworks. Bioinformatics. 2018 Sep 1;34(17):i972-i980. doi: 10.1093/bioinformatics/bty613. PMID: 30423088; PMCID: PMC6129270.
